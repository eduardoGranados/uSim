# Simulador de Redes: uSim

=======

Simulador de Redes por eventos Discretos creado en la Universidad de los Andes, Mérida Venezuela.


## Requerimientos

Sistema Operativo: Desarrollado y probado en Linux. Corre en Mac OS con pequeños bugs que serán arreglados próximamente.
Compilador de C++, recomendado g++ de [GCC](https://gcc.gnu.org/)
Librería [QT](https://qt-project.org/downloads) para C++
